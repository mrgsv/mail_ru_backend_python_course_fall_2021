## HW#1: Tic-Tac-Toe game.

In order to start Tic Tac Toe game one should use:
```shell
python3 t3_game.py -n <board_size>
```
Default board_size = 3.

Board looks like this:
```shell
    0   1   2 
  +---+---+---+
0 |   |   |   |
  +---+---+---+
1 |   |   |   |
  +---+---+---+
2 |   |   |   |
  +---+---+---+
```
You can play like this:
```shell
 Player#1, your turn:
 1 1  
    0   1   2 
  +---+---+---+
0 |   |   |   |
  +---+---+---+
1 |   | X |   |
  +---+---+---+
2 |   |   |   |
  +---+---+---+
```
